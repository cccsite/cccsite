# C 語言

C 是速度很快的語言，1970 年和 UNIX 作業系統一同被開發出來，構建了整個電腦工業的底層程式，這類程式通常稱為《系統程式》，包含《作業系統、編譯器、組譯器、虛擬機》等領域，都是 C 語言專長的範圍。

相較於 C++，C 語言簡單而乾淨，但是由於威力強大的指標是 C 語言的特色，也因此常會造成指標誤用的情況。

C/C++/Rust 這三個語言都屬於速度很快的《底層語言》，適合用來開發 [系統程式](../課程/系統程式/) 。
