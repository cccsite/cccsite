# C++ 語言

C++ 是具有物件導向語法的 C 語言相容版，標準函式庫比 C 語言豐富，同樣也具有指標，也是有點危險又易出錯的語言。

C/C++/Rust 這三個語言都屬於速度很快的《底層語言》，適合用來開發 [系統程式](../課程/系統程式/) 。
